# Password Reset Workflow

## Issue
User is unable to log in due to forgotten or expired password.

## Troubleshooting Steps
1. Verify user identity using approved verification questions.
2. Check account status in Active Directory.
3. Reset password following security policy.
4. Ensure user creates a strong password.
5. Confirm successful login and update ticket.

## Root Cause
Password expired or forgotten.

## Resolution
Password reset and user successfully logged in.
